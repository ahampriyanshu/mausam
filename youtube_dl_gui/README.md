# youtube_dl_gui

# Installation

## Linux
```bash
cd ~
wget -q https://github.com/ahampriyanshu/scripts_101/files/5870156/yt_dl_gui.tar.gz
tar -xzvf yt_dl_gui.tar.gz && cd yt_dl_gui
mv yt_dl_gui.desktop  ~/.local/share/applications && cd ..
sudo mv yt_dl_gui /usr/bin
```
